# DataMint Blog Post -- Gemini Image Generation Prompts

> Style guide applied to all images: Modern, clean, minimalist tech aesthetic. Dark theme using deep purple `#2d1b69` as the primary background color and mint green `#00e676` as the accent/highlight color. White `#FFFFFF` text and line art. Flat design / isometric style -- no stock photo feel. Data visualization elements such as charts, grids, and flowing data streams. Consistent visual language across all seven images.

## Usage Notes

- **Generation tool:** These prompts are written for Google Gemini's image generation model (Imagen 3 or later). They may need minor adjustments for other models.
- **Aspect ratios:** Hero image is 1200x630 (approximately 1.9:1, suitable for Open Graph / social media preview). Section images are 800x500 (1.6:1, fits well within Medium's content width).
- **Color consistency:** Every prompt enforces the same palette -- deep purple `#2d1b69`, mint green `#00e676`, white `#FFFFFF`, with muted grays and contextual accent colors (red for severity/errors, yellow for warnings). If the model drifts on color, re-emphasize the hex codes in a shorter follow-up prompt.
- **Iteration tip:** If a generated image is too busy, append "Simplify the composition. Reduce the number of elements. More whitespace." to the prompt. If it is too plain, append "Add subtle data-stream particle effects in mint green flowing across the background."
- **Format:** Generate as PNG for lossless quality. Compress with a tool like TinyPNG before uploading to the blog if file size is a concern.
- **Accessibility:** Every image has an alt text field above. Use these as the `alt` attribute in `<img>` tags or the alt text field in your CMS. Screen readers rely on these descriptions.
