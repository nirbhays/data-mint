# DataMint Roadmap

## Vision
Generating test data for LLM pipelines should be instant, free, and reproducible — no API keys required ever.

## ✅ Shipped
- QA pair generation (question + answer datasets)
- Adversarial / red-team prompt generation
- Structured tabular data generation
- 1,000+ items/second
- Fully deterministic with seed control
- 37+ tests

## 🔨 In Progress
- [ ] Multilingual dataset generation (Spanish, French, German)
- [ ] pytest fixture plugin (`pytest-datamint`)
- [ ] HuggingFace datasets format export

## 📋 Planned — Q2 2025
- [ ] Domain-specific templates (medical, legal, finance)
- [ ] Conversation / multi-turn dataset generation
- [ ] JSONL and Parquet export formats
- [ ] CLI dataset previewer

## 📋 Planned — Q3 2025
- [ ] LLM-assisted generation mode (optional, for higher quality)
- [ ] Dataset quality scoring
- [ ] Integration with ModelLedger for eval lineage tracking

## 💡 Under Consideration
- Web UI for no-code dataset generation
- OpenAI evals format export
- RAG evaluation dataset templates
